# Writing An Interpreter



# Getting Started

### Enter the REPL

`$ go run ./src/main`

### Debug Mode

Append `-d` argument to show the shape of the AST while parsing.

`$ go run ./src/main -d`

## Components

1. Lexer
2. Parser
3. Evaluator
4. REPL ( **R**ead **E**valuate **P**rint **L**oop )
5. Builtin Function
